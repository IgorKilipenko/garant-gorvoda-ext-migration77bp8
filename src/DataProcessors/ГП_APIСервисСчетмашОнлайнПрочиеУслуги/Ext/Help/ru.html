По кнопке "Заполнить".

     При заполнении табличной части выполняется проверка, указан ли в документе "Регистрация оплаты",
в табличной части "Оплата по лицевым счетам" платежный документ в поле "Документ оплаты".

     "Документ оплаты" может быть документом "Поступление на расчетный счет" или "Поступление наличных" (ПКО).

     Если не указан  выводится сообщение:
 "В документе № , не указан платежный документ. Укажите платежный документ и выполните заполнение заново."
